# comp248-gaming-project

## Project title
What Makes a Successful Video Game?
A Data-Driven Analysis of player Reception and Commercial Performance

## Overview
This Project analyzes video game sales, ratings and Steam metadata to identify patterns associated with successful games.

## Objectives
- Identify attributes linked to high video game sales
- Exlpore the relationship between ratings and commercial success
- Build a simple machine learning classfier
- Create interactive Power BI dashboards

## Datasets
- Video Game Sales 2024 Dataset
- Metacritic Games Dataset
- Steam Games Dataset

## Tools
- Python
- Pandas
- Numpy
- Scikit-learn
- Jupyter Notebook
- Power BI
